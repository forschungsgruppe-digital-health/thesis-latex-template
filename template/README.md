# Thesis starter

A minimal fill-in-the-blanks thesis. Edit [`thesis.tex`](thesis.tex) (placeholders are
marked « … »), add references to [`references.bib`](references.bib), and put images in
[`figures/`](figures/).

- **Compile:** pdfLaTeX + BibTeX — `latexmk thesis.tex`, or just Build in the dev container.
- **Language:** German by default; add the `en` class option for English.
- **Thesis type:** swap `\bachelortitlepage` for `\mastertitlepage` /
  `\diplomatitlepage` / `\dissertationtitlepage` (same 8 arguments).

Full guide: [../docs/filling-in.md](../docs/filling-in.md).
